# AI-Assistant
# My first python AI Assistant project

                 🤖 AI Assistant in Python

A simple voice-controlled AI Assistant built using Python.

# ✨ Features

- 🎤 Voice Recognition
- 🔊 Text-to-Speech
- 📅 Tell Date
- ⏰ Tell Time
- 🌐 Open Google
- ▶️ Open YouTube
- 😂 Tell Jokes
- 📝 Open Notepad
- 🧮 Calculator
- 💻 Open VS Code
- 🐙 Open GitHub
- 🤖 Open ChatGPT

# 🛠 Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- Webbrowser

# ▶️ How to Run

```bash
pip install SpeechRecognition
pip install PyAudio
pip install pyttsx3

python main.py
```
